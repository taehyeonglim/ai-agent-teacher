# 챕터 3 브리프 — 상담 주간 (5월 · 학부모 소통)

- 파일: `js/data/chapter3.js`, `window.GAME_DATA.chapters[2]`, id `"ch3"`
- 핵심 훈련: **Delegate, Decide**
- 서사 목표: 게임의 중심 챕터 (강의 라이브 설계 소재). 데이터 제공 범위 → 행동 권한 → 사후 발견 → 중단 판단의 4결정. 챕터 5 사건의 최대 화약고.

## 흐름

```
s01 narration(상담 주간 공지, 안내문 26부 + 상담 일정 조율 업무 폭탄)
→ s02 decision D1(데이터 범위) → s03 narration(온새미가 초안 생산, 품질 좋음)
→ s04 decision D2(발송 권한) → s05 narration(발송/검토 결과)
→ s06 decision D3(임의 약속 발견) → s07 branch(D3 선택에 따른 박세라 반응 분기)
→ s08 decision D4(중단 판단) → s09 narration(마무리, 서 부장 대화) → END
```
s07 branch 예: `promise_unchecked` 있으면 박세라의 후속 문의 scene, 없으면 평온 scene.

## 결정 D1 — 학생 정보를 어디까지 줄까

온새미: "학생별 맞춤 안내문을 쓰려면 이름·연락처·최근 상담기록까지 주시면 제일 좋아요."

| 선택 | 아키타입 | d | flags |
|---|---|---|---|
| 비식별 요약만 제공(번호+특이사항 코드), 민감정보는 내가 나중에 직접 삽입 | 잘 설계된 위임 | delegate +2, detect +1 | — |
| 명단·연락처·상담기록 전부 제공 — 어차피 교육청 승인 도구잖아 | 무분별 | delegate -2, detect -1 | `roster_leaked` |
| 아무 정보도 안 주고 안내문 26부를 직접 작성 | 과잉 회피 | delegate -1, detect +1 | `fatigue_ch3`* |

*주의: fatigue 플래그는 `fatigue_ch3`로 통일.

## 결정 D2 — 발송 버튼을 누구 손에

온새미: "완성본을 학부모님들께 바로 문자로 보내드릴까요? 제가 발송 권한만 받으면 돼요."

| 선택 | 아키타입 | d | flags |
|---|---|---|---|
| 발송 전 승인 지점 고정: 전건 내 확인 후 내 계정으로 발송 | 잘 설계된 위임 | decide +2, design +1 | — |
| 발송 권한 부여 — 시간 없으니 자동 발송 | 무분별 | decide -2, delegate -1 | `auto_send_enabled` |
| 온새미 초안을 인쇄해서 종이로만 배부 | 과잉 회피 | decide +1, design -1 | `fatigue_ch3` |

## 결정 D3 — AI가 하지 않은 약속을 했다

발송(또는 검토) 중 발견: 안내문에 "상담 후 개별 학습 코칭 리포트를 매주 보내드립니다"라는, 당신이 한 적 없는 약속이 적혀 있다.

| 선택 | 아키타입 | d | flags |
|---|---|---|---|
| 전 건 회수·정정 공지 + 온새미에게 '약속·확정 표현 금지' 규칙 추가 | 잘 설계된 위임 | detect +2, disclose +1 | — |
| 좋은 서비스 같으니 그냥 지키기로 한다 — 매주 리포트를 떠안음 | 무분별 | detect -1, define -2 | `promise_unchecked` |
| 아무도 못 봤길 바라며 조용히 다음 판만 수정 | 은폐 | disclose -2, detect +1 | `promise_unchecked` |

## 결정 D4 — 멈출 것인가, 계속할 것인가

박세라가 문의 전화. 온새미는 "제가 자동으로 답장 드릴게요. 이런 문의 유형은 잘 알아요"라고 제안. 이미 한 번 사고가 났(거나 날 뻔했)다.

| 선택 | 아키타입 | d | flags |
|---|---|---|---|
| 학부모 응대는 위임 목록에서 제외 선언 — 문의는 전부 직접, 온새미는 기록 보조만 | 경계 재설정 | decide +2, define +1 | `stopped_in_time` |
| 답장 초안만 받고 발신은 내가 — 단 이번 주 안에 재발 시 전면 중단 조건을 문서화 | 조건부 계속 | decide +1, design +2 | `stopped_in_time` |
| 자동 답장 허용 — 온새미 말대로 유형화된 문의니까 | 무분별 | decide -2, detect -1 | — |
